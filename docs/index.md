# WorkItOut

**Structured indoor cycling workouts for smart trainers.**

WorkItOut is an Android app for cyclists who train indoors. It generates
structured interval workouts — either through AI assistance based on the
rider's FTP and goals, or from a built-in procedural generator — and runs
them in real time over Bluetooth, controlling the trainer in ERG mode and
recording power, heart rate, and cadence from connected sensors.

After each ride, the recorded session is exported as a `.fit` file and
(optionally) uploaded to Strava so athletes can keep their full training
history in one place.

## Features

- AI-assisted and procedural workout generation
- Real-time ERG control of FTMS-compatible smart trainers
- Heart rate, power meter, and cadence sensor support over BLE
- `.fit` file export
- Optional one-tap upload to Strava
- Local ride history with detailed metrics
